# LAA Narrative Generator - User Manual
## Simple Guide for Woodruff Billing Staff

---

## What This Tool Does

The LAA Narrative Generator transforms solicitor submissions into professional narratives for the Legal Aid Agency (LAA). It takes the PDF that solicitors create using the Uplift Collator and converts it into a compelling narrative that supports their uplift claim.

---

## How It All Works Together

### The Complete Process:

1. **Step 1: Solicitor Uses Uplift Collator**
   - Solicitor opens the Uplift Collator in their web browser
   - They fill in the form with case details and justifications
   - They click "Generate PDF" and save it
   - They email this PDF to Woodruff Billing

2. **Step 2: You Use the Narrative Generator**
   - You receive the solicitor's PDF
   - You open the Narrative Generator
   - You load their PDF
   - The tool creates a professional narrative

3. **Step 3: Final Narrative**
   - The narrative includes all LAA references
   - It uses the solicitor's exact explanations
   - You save it as a Word document
   - You can then edit and submit to LAA

3. Wait for installation to complete
4. Press any key when prompted

### Daily Use
1. Double-click `run.bat`
2. The application window will open
3. You'll see "Version 0.1.1" at the top

## Using the Application

### Loading a PDF

1. **Click "Browse PDF"**
   - A file browser will open
   - Navigate to where you saved the solicitor's PDF
   - Select the PDF file
   - Click "Open"

2. **Check the filename**
   - The filename will appear in the white box
   - Make sure it's the correct PDF

### Generating the Narrative

1. **Click "Generate Narrative"**
   - The green button will process the PDF
   - Wait a few seconds
   - The narrative will appear in the text area

2. **Review the narrative**
   - Check the case details are correct
   - Verify the uplift percentage
   - Ensure all explanations are included

### Saving Your Work

You have three options:

#### Option 1: Save as Word Document (Recommended)
1. Click **"Save as Word Document"**
2. Choose where to save the file
3. Give it a meaningful name
4. Click "Save"
5. The document will have:
   - Arial Nova Cond Light 12pt font
   - Bold headings
   - Proper spacing
   - Professional formatting

#### Option 2: Copy to Clipboard
1. Click **"Copy to Clipboard (for Word)"**
2. Open Microsoft Word
3. Press Ctrl+V to paste
4. The formatting will be preserved

#### Option 3: Save as Text File
1. Click **"Save as Text File"**
2. Choose location and name
3. This creates a plain text version

## Understanding the Output

### What the Narrative Contains

The generated narrative includes:

1. **Title Section**
   - "Claim For Additional General Enhancement"
   - Case name, matter type, fee earner

2. **Introduction**
   - States the uplift percentage
   - References CPR 44.4(3)
   - Mentions LAA Costs Assessment Guidance

3. **Panel Membership** (if applicable)
   - Lists any panel memberships
   - References CAG Sections 12.20-12.23

4. **Stage 1: Threshold Test**
   - Lists exceptional factors
   - Includes solicitor's explanations
   - References specific CAG sections

5. **Stage 2: Level Justification**
   - Supports the percentage claimed
   - Additional factors and explanations
   - More CAG references

6. **Conclusion**
   - Summarises the justification
   - States uplift is "considered and justifiable"

### Important Points

- **No Changes to Solicitor's Words**: The tool uses exactly what the solicitor wrote
- **Automatic LAA References**: Adds proper section numbers automatically
- **Professional Structure**: Organises information clearly
- **Ready for Submission**: Output is formatted for LAA requirements

## Common Tasks

### Processing Multiple PDFs

1. Process first PDF
2. Save the narrative
3. Click "Clear" button
4. Load next PDF
5. Repeat process

### Checking What Was Selected

Look for bullet points (•) in the narrative:
- Each bullet is something the solicitor ticked
- The text after is their explanation
- Empty explanations are excluded

### Finding the Uplift Percentage

The percentage appears in:
- The introduction paragraph
- The Stage 2 section
- The conclusion

## Troubleshooting

### PDF Won't Load
- Check it's a PDF from the Uplift Collator
- Try saving the PDF to your desktop first
- Close and reopen the application

### No Narrative Generated
- Check the PDF has content
- Ensure it's from the current version of Uplift Collator
- Try clicking "Clear" then reload

### Buttons Don't Work
- Wait for current process to complete
- Check a PDF is loaded
- Try restarting the application

### Font Looks Wrong in Word
- The tool tries to use Arial Nova Cond Light
- If not installed, it uses regular Arial
- You can change the font in Word after saving

## Tips for Best Results

1. **Check PDFs First**
   - Open the solicitor's PDF before processing
   - Ensure all sections are filled
   - Verify the uplift percentage is stated

2. **Review Before Saving**
   - Read through the generated narrative
   - Check all solicitor explanations are included
   - Verify LAA references make sense

3. **File Organisation**
   - Create folders by date or solicitor
   - Use clear filenames
   - Keep original PDFs with narratives

4. **Quality Control**
   - The tool doesn't add or interpret
   - If explanations seem generic, that's what solicitor provided
   - You may need to request more detail from solicitor
