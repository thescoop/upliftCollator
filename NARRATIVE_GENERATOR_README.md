# LAA Uplift Narrative Generator
## Internal Tool for Woodruff Billing Ltd

### Purpose
This tool converts solicitor PDF submissions from the Uplift Collator into professional LAA-compliant narratives with proper threshold references.

### Features
- Extracts all form data from solicitor's PDF
- Generates narratives that reference specific LAA thresholds
- Uses solicitor's exact explanations (no embellishment)
- States uplift percentage as "considered and justifiable"
- Exports to plain text file

### Setup Instructions

#### Option 1: Run from Python
1. Ensure Python 3.7+ is installed on your computer
2. Double-click `setup_narrative_generator.bat` to install dependencies
3. Run the application with: `python narrative_generator.py`

#### Option 2: Create Standalone Executable
1. Complete Option 1 setup first
2. Double-click `build_exe.bat` to create the executable
3. Find the executable in the `dist` folder
4. You can move this .exe file anywhere and run it without Python

### How to Use

1. **Start the Application**
   - Run `narrative_generator.py` or the executable

2. **Load a PDF**
   - Click "Browse PDF" 
   - Select the solicitor's PDF (generated from the Uplift Collator)

3. **Generate Narrative**
   - Click "Generate Narrative"
   - The narrative will appear in the text area

4. **Review the Narrative**
   - Check that all solicitor inputs are correctly captured
   - Verify LAA threshold references are appropriate

5. **Save the Narrative**
   - Click "Save as Text File"
   - Choose location and filename
   - Default format: uplift_narrative_YYYYMMDD_HHMMSS.txt

### What the Narrative Includes

- **Case Information**: Fee earner, matter type, case name
- **Panel Membership**: If applicable, with CAG Section 12.20-12.23 reference
- **Stage 1 Threshold Test**: With references to:
  - CAG Sections 12.8.1-12.8.3
  - Spec Para 6.13(a)-(c)
- **Stage 2 Enhancement Level**: With references to:
  - CAG Sections 12.9(a)-(c)
  - Spec Para 6.15
- **Conclusion**: States uplift as "considered and justifiable"

### Important Notes

- The tool ONLY uses information provided by the solicitor
- No additional facts or arguments are added
- All explanations are the solicitor's exact words
- The narrative structure provides the professional framework

### Troubleshooting

**PDF won't load:**
- Ensure it's a PDF generated by the Uplift Collator
- Check the PDF isn't corrupted

**Missing data in narrative:**
- The solicitor may not have filled all sections
- Check the original PDF for completeness

**Installation issues:**
- Ensure Python 3.7+ is installed
- Run `pip install pdfplumber` manually if needed
